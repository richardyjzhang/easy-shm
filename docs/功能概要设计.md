# 结构健康监测系统 — 功能概要设计

## 1 系统概述

结构健康监测系统（Structural Health Monitoring System, SHM）用于对桥梁、隧道、建筑等工程结构进行长期在线监测，实现数据采集、报警处置、档案管理等核心业务。

## 2 功能模块

### 2.1 综合管理

| 子菜单   | 说明                                                   |
| -------- | ------------------------------------------------------ |
| 工作台   | 系统首页，使用二维地图展现结构物分布和统计             |
| 综合查询 | 跨模块的统一检索入口，支持按结构、时间、指标等条件查询 |
| BIM协同  | 基于 BIM 模型可视化展示结构状态与监测点位              |

### 2.2 基本信息

| 子菜单   | 说明                                                 |
| -------- | ---------------------------------------------------- |
| 结构管理 | 维护被监测结构的基本信息（名称、类型、位置、参数等） |
| 档案管理 | 管理结构相关的文档资料（设计图纸、检测报告等）       |

### 2.3 监测管理

| 子菜单       | 说明                                               |
| ------------ | -------------------------------------------------- |
| 数据查看     | 按监测项、时间范围查看采集数据，包括实时值和历史值 |
| 报警处置     | 报警事件列表，支持查看详情、处置等操作             |
| 监测项管理   | 管理监测项（指标）及其下属的监测内容（物理量）     |
| 监测设备管理 | 管理传感器、采集仪等硬件设备的台账信息与在线状态   |
| 报警阈值管理 | 为各监测项设置多级报警阈值                         |

### 2.4 系统配置

| 子菜单   | 说明                                     |
| -------- | ---------------------------------------- |
| 机构管理 | 维护组织机构信息                         |
| 用户管理 | 用户账号的增删改查、角色分配             |
| 个性化   | 用户级别的界面偏好设置（主题色、布局等） |

## 3 权限体系

### 3.1 角色定义

系统设置**三类**角色；除下表特别说明外，业务权限主要体现在**数据可见与操作范围**（按机构隔离）上：

| 角色       | 说明                                                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| 系统管理员 | 可访问全部功能；可查看、操作**全部机构**下的业务资源；机构管理、用户管理均为**全局**范围。                                           |
| 机构管理员 | 在本机构范围内拥有与业务相关的完整操作权限：可查看、新增、编辑、删除**本机构**下的各类业务资源；**用户管理**可管理本机构内全部用户。 |
| 机构用户   | 可查看、操作**本机构**下的常规业务资源（结构、监测、档案等）；**用户管理**场景下仅能维护**本人账号**（如个人信息），不可管理他人。   |

### 3.2 机构模型

- 机构之间**不设级联**关系，各机构平级并列、互相独立。
- 每个用户归属于一个机构。
- 机构是资源隔离的基本单元。

### 3.3 资源与机构关联

除机构本身外，系统中所有业务资源均关联到某一机构：

| 资源类型        | 与机构关系                 |
| --------------- | -------------------------- |
| 结构物          | 归属于创建它的机构         |
| 监测项          | 归属于创建它的机构         |
| 监测内容        | 随所属监测项归属机构       |
| 传感器/采集设备 | 随所监测的结构物归属机构   |
| 报警事件        | 随触发报警的结构物归属机构 |
| 档案资料        | 随所关联的结构物归属机构   |

### 3.4 数据权限规则

| 操作场景           | 系统管理员         | 机构管理员         | 机构用户               |
| ------------------ | ------------------ | ------------------ | ---------------------- |
| 查看资源列表       | 全部机构的资源     | 仅本机构的资源     | 仅本机构的资源         |
| 新增/编辑/删除资源 | 任意机构           | 仅本机构           | 仅本机构               |
| 工作台统计与地图   | 汇总所有机构数据   | 仅本机构数据       | 仅本机构数据           |
| 机构管理           | 可增删改查所有机构 | 不可见             | 不可见                 |
| 用户管理           | 可管理所有用户     | 可管理本机构内用户 | 仅本人（不可管理他人） |
